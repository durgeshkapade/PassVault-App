# Password Manager

## Features

- **Store Passwords:** Safely store your passwords in an encrypted database.
- **Create Passwords:** Generate and save new passwords.
- **Update Passwords:** Edit your existing passwords.
- **Delete Passwords:** Remove passwords that you no longer need.
- **Password Generation:** Create strong, secure passwords.
- **Categorization:** Organize passwords into categories for easy access.
- **Search Functionality:** Quickly find passwords using the search feature.

## Technologies Used

- **Frontend:** React
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Other Libraries:** 
  - bcryptjs: Password hashing
  - JWT: Authentication
  - React Router: Navigation

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- [npm](https://www.npmjs.com/get-npm)
- [MongoDB](https://www.mongodb.com/) (local or cloud-based)
